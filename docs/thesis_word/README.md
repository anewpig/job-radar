# Thesis Word Handoff

這個目錄放目前可直接開啟的 Word 主稿檔：

- `ai_thesis_master_draft.docx`

## 來源

這份檔案由：

- `../ai_thesis_master_draft.md`

透過 macOS 內建 `textutil` 轉出。

## 目前狀態

- 章節內容已完整帶入
- 適合直接在 Word 中接手排版
- 引用已在文內以作者年格式寫入
- 圖檔位置已在 Markdown / LaTeX 主稿中對齊

## 建議接手順序

1. 先在 Word 打開 `ai_thesis_master_draft.docx`
2. 套用學校論文格式
3. 補：
   - 作者
   - 指導教授
   - 系所
   - 日期
4. 將 Chapter 2 文獻引用補齊成正式格式
5. 將兩張 SVG 圖轉為最終論文插圖樣式

## 備註

由於 `textutil` 是無依賴快速轉出，標題層級與表格仍建議在 Word 內再做一次版面整理。
